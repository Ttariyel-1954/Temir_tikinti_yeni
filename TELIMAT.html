🔧 Addım-addım qurulum

R paketlərini quraşdırın

RStudio konsolunda bu əmri icra edin:

R install.packages(c( "shiny", "bslib", "DBI", "RPostgres", "pool", "DT", "dplyr", "plotly", "httr2", "jsonlite", "shinyjs", "shinycssloaders", "openxlsx" ))

PostgreSQL bazasını hazırlayın

temir_tikinti bazası yaradılmış və 21 cədvəl ilə doldurulmuş olmalıdır.

Terminal psql -d temir_tikinti -c "\dt" # cədvəlləri yoxlayın

.env faylını konfiqurasiya edin

.env.example faylını .env adı ilə kopyalayın və dəyərləri daxil edin:

.env # AI Model ANTHROPIC_API_KEY=sk-ant-api03-... # PostgreSQL Database DB_HOST=localhost DB_PORT=5432 DB_NAME=temir_tikinti DB_USER=your_user DB_PASSWORD=your_password

Tətbiqi işə salın

Variant A — RStudio: app.R faylını açın, yuxarıdakı "Run App" düyməsini basın.

Variant B — Terminal:

Terminal cd ~/Desktop/Təmir_tikinti_Yeni/agent R -e 'shiny::runApp("app.R", port=5050, launch.browser=TRUE)'

Brauzer avtomatik açılacaq: http://127.0.0.1:5050

💡
Qeyd: Lokal istifadədə DB_USER və DB_PASSWORD boş qala bilər — sistem defolt sistem istifadəçisi ilə qoşulacaq.

AI Köməkçi — söhbət paneli

🤖 AI ilə necə işləmək?

Sağ paneldəki AI Köməkçi Claude Sonnet 4 modeli ilə işləyir. Bazadakı 21 cədvəldən real vaxtda məlumat çəkərək Azərbaycan dilində ətraflı cavab verir.

API açarını daxil edin

Başlıq panelinin sağında "API açarı..." sahəsinə Anthropic API açarınızı daxil edin. .env faylında qeyd olunubsa, avtomatik doldurulacaq.

Sualınızı yazın və ya hazır sual seçin

Alt hissədəki yazı sahəsinə öz sualınızı yazın, Enter basın və ya "Göndər" düyməsini klikləyin.

AI cavab hazırlayır

AI bazaya sorğu göndərir, məlumatı alır, təhlil edir və HTML5 formatında nəfis cavab qaytarır. Bu proses 5-20 saniyə çəkə bilər.

Cavabı oxuyun və ya yadda saxlayın

Cavabda cədvəllər, siyahılar, diaqramlar ola bilər. "💾 Yadda saxla" düyməsi ilə HTML5 hesabat kimi saxlayın.

🔧 AI-ın 14 aləti

AI Köməkçi bazadan məlumat almaq üçün 14 xüsusi alətə malikdir. Sual verəndə hansı aləti istifadə edəcəyinə özü qərar verir:

Cavabları və cədvəlləri yadda saxlama

💾 AI cavablarını HTML5 kimi saxlama

Chat panelinin yuxarısındakı "💾 Yadda saxla" düyməsini basın.

Nə baş verir?

  • Bütün söhbət tarixçəsi (suallar + cavablar) nəfis HTML5 sənədi kimi saxlanır
  • Fayl Cavablar/ qovluğunda yaradılır
  • Fayl adı: Cavab_2026-03-15_14-30-00.html
  • Faylı brauzerdə açıb oxuya, çap edə bilərsiniz

HTML faylının xüsusiyyətləri:

  • Gradient başlıq — İdarə loqosu və tarix
  • Suallar mavi fonda, cavablar ağ fonda
  • Cədvəllər, siyahılar, kodlar — tam stilizə edilmiş
  • Çap üçün optimallaşdırılmış (Ctrl+P)

📊 Cədvəlləri Excel-ə yükləmə

Data panelinin üstündəki "📊 Excel-ə yüklə" düyməsini basın.

Nə baş verir?

  • Hazırda aktiv olan tab-ın cədvəli .xlsx faylı kimi saxlanır
  • Fayl Excel/ qovluğunda yaradılır
  • Fayl adı: Layiheler_2026-03-15_14-30-00.xlsx
  • Microsoft Excel, LibreOffice, Google Sheets-də açıla bilər

Excel faylının xüsusiyyətləri:

  • Başlıq sətri — mavi fon, ağ hərflər, qalın yazı
  • Cüt sətirlərdə boz fon (zebra effekti)
  • Avtomatik sütun genişliyi
  • Filtr aktiv — hər sütunda filtr düyməsi
  • Başlıq sətri dondurulmuş (freezePane)
📁
Qovluq yeri: Fayllar tətbiqin işlədiyi qovluqda (agent/Cavablar/ və agent/Excel/) yaradılır. .gitignore-da bu qovluqlar əlavə edilməyib — istəsəniz, əlavə edə bilərsiniz.

Tez-tez verilən suallar (FAQ)

❓ API açarı olmadan sistem işləyirmi?
Dashboard, cədvəllər, qrafiklər və xəbərdarlıqlar API açarsız da işləyir. Yalnız AI Köməkçi söhbəti üçün Anthropic API açarı lazımdır.
❓ AI bazaya zərər verə bilərmi?
Xeyr. AI yalnız SELECT sorğuları icra edə bilər. INSERT, UPDATE, DELETE əmrləri kod səviyyəsində bloklanıb.
❓ Cavablar hansı dildədir?
AI həmişə Azərbaycan dilində cavab verir. System prompt-da bu qaydalar təyin olunub.
❓ AI cavabı nə qədər gözləmək lazımdır?
Sadə suallar üçün 5-10 saniyə, mürəkkəb təhlillər üçün 15-25 saniyə. Bu müddətdə "Düşünürəm..." animasiyası görünür.
❓ Eyni vaxtda neçə nəfər istifadə edə bilər?
Lokalde bir nəfər. Shinyapps.io-da pulsuz planda 5 aktiv bağlantı, ödənişli planlarda daha çox.
❓ Excel və HTML faylları harada saxlanır?
Tətbiqin işlədiyi qovluqda — agent/Cavablar/ və agent/Excel/ içində. Shinyapps.io-da bu funksiya serverdə yazır (müvəqqəti).
❓ Başqa AI model istifadə etmək olarmı?
app.R faylında call_claude funksiyasındakı model parametrini dəyişin. Məsələn: "claude-opus-4-6" (daha güclü, daha bahalı) və ya "claude-haiku-4-5-20251001" (daha sürətli, daha ucuz).
❓ Yeni cədvəl əlavə etsəm nə etməliyəm?
1) Bazaya cədvəli yaradın, 2) app.R-da yeni sorğu funksiyası yazın, 3) AI_TOOLS siyahısına yeni alət əlavə edin, 4) execute_ai_tool-a əlavə edin, 5) UI-da yeni tab yaradın.
❓ Telefondan istifadə etmək olarmı?
Bəli. İnterfeys responsiv dizayn edilib — kiçik ekranlarda panellər alt-alta düzülür, sürüşdürmə çubuğu gizlənir.
❓ Söhbəti necə sıfırlayım?
Chat panelinin yuxarısındakı "Yeni söhbət" düyməsini basın. Bütün tarixçə silinir və yeni sessiya başlayır.
